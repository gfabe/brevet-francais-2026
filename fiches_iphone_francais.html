1Grammaire : nature & fonction›

Natures (classes de mots)

Variables : nom, déterminant, adjectif, pronom, verbe. Invariables : adverbe, préposition, conjonction, interjection.

✔ La nature = ce qu'un mot est ; la fonction = le rôle qu'il joue.

Fonctions

Sujet · COD (qui/quoi) · COI (à qui) · attribut (après être) · épithète · CC (temps, lieu, manière).

Propositions subordonnées

2Conjugaison & temps›

Modes : indicatif (réel) · subjonctif (il faut que…) · conditionnel (hypothèse, politesse) · impératif (ordre).

Imparfait vs passé simple

ImparfaitPassé simple
décor, durée, habitudeaction soudaine, 1ᵉʳ plan
Ex : « Il dormait quand le téléphone sonna. »

Présent : énonciation · vérité générale · narration · habitude.

⚠ « Si » + imparfait → conditionnel. Jamais « si » + conditionnel.
3Orthographe & homophones›

Participe passé

  • avec être → accord avec le sujet
  • avec avoir → accord avec le COD placé avant
  • avoir sans COD avant → invariable

Homophones — test de remplacement

a / àa = avait
et / estest = était
son / sontsont = étaient
on / ontont = avaient
ces / sesceux-là / les siens
ou / oùou = ou bien
✔ En dictée : à chaque homophone, remplace dans ta tête pour trancher.
4Figures de style›
  • Comparaison : avec outil (comme, tel)
  • Métaphore : sans outil
  • Personnification : une chose agit comme un humain
  • Hyperbole (exagère) · litote (dit moins) · euphémisme (adoucit)
  • Antithèse (idées opposées) · oxymore (« nuit blanche »)
  • Anaphore (répétition début) · gradation · métonymie
  • Allitération (consonnes) · assonance (voyelles)
✔ Nomme la figure puis explique son effet (insister, choquer, embellir…).
5Registres (tonalités)›
  • Comique : faire rire · ironique : dire le contraire
  • Pathétique : pitié · tragique : destin fatal
  • Lyrique : sentiments · épique : exploits
  • Fantastique : doute réel/surnaturel · polémique : attaquer
Repère : champ lexical dominant + ponctuation (! ?) + figures.
6Analyse du récit›

Points de vue

  • Interne : on voit par un personnage (ses pensées)
  • Externe : caméra neutre
  • Omniscient : le narrateur sait tout

8Poésie & versification›
  • Alexandrin = 12 syllabes · décasyllabe = 10 · octosyllabe = 8
  • Le -e muet compte sauf en fin de vers / devant voyelle

Rimes & strophes

Plates AABB · croisées ABAB · embrassées ABBA. Quatrain (4), tercet (3), sonnet = 14 vers.

Enjambement · césure (6//6) · hémistiches.

✔ Relie toujours la forme (rythme, sons) au sens (émotion, idée).
★Méthodes — les 3 épreuves›

Compréhension

J'affirme + je cite (« … ») + j'explique. Jamais de citation seule.

Dictée

1) accorde les verbes · 2) teste les homophones · 3) vérifie les participes passés.

Rédaction — réflexion

Intro (thèse) → 2-3 arguments + exemples → conclusion. Connecteurs : d'abord, de plus, cependant, donc.

Rédaction — invention

Respecte les consignes, reste cohérent avec le texte, soigne dialogues & temps.

✔ L'orthographe est notée dans la rédaction — garde 5 min pour relire.
