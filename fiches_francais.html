Grammaire : nature & fonction

classes de mots · fonctions · propositions
🏷️Les classes grammaticales (nature)

Variables : nom, déterminant, adjectif, pronom, verbe.

Invariables : adverbe, préposition, conjonction, interjection.

✔ La nature = ce qu'un mot est (toujours la même). La fonction = le rôle qu'il joue dans la phrase (varie).
🎯Les fonctions essentielles
  • Sujet : qui fait l'action ? (« Léa court »)
  • COD : qui ? quoi ? sans préposition
  • COI : à qui ? à quoi ? (préposition)
  • Attribut du sujet : après être, sembler, paraître…
  • Épithète : adjectif collé au nom
  • CC : temps, lieu, manière, cause…
🧱Phrase simple / complexe

1 verbe conjugué = 1 proposition.

  • Simple : un seul verbe conjugué
  • Complexe : plusieurs propositions
Liens : juxtaposition ( , ; : ), coordination (mais, ou, et, donc, or, ni, car), subordination.
🔗Les propositions subordonnées
TypeIntroduite par
Relative (complète un nom = antécédent)qui, que, dont, où…
Conjonctive CODque
Circonstanciellequand, parce que, si, pour que…
Ex : « Le livre que tu lis » → relative, COD de l'antécédent livre.

Conjugaison & valeurs des temps

modes · temps · valeurs dans le récit
🎭Les modes
  • Indicatif : le réel
  • Subjonctif : souhait, doute, ordre (après que, il faut que)
  • Conditionnel : hypothèse, politesse
  • Impératif : ordre
  • Non personnels : infinitif, participe, gérondif
⏳Imparfait vs passé simple
ImparfaitPassé simple
décor, descriptionaction soudaine
habitude, durée1ᵉʳ plan du récit
arrière-planfait ponctuel
Ex : « Il dormait (imp.) quand le téléphone sonna (p.s.). »
🕐Valeurs du présent
  • D'énonciation : au moment où l'on parle
  • De vérité générale : « L'eau bout à 100 °C »
  • De narration : rend un récit vivant
  • D'habitude : « Chaque jour, il marche. »
🔮Subjonctif & conditionnel

Subjonctif : « Il faut que tu viennes », « Je veux qu'il parte ».

Conditionnel : hypothèse (« Si j'avais le temps, je viendrais »), futur dans le passé, politesse.

⚠ « Si » n'est jamais suivi du conditionnel : Si j'avais (imparfait) → je viendrais.

Orthographe & homophones

accords · participe passé · homophones (spécial dictée)
🔁Accord du participe passé
  • avec être → s'accorde avec le sujet (« Elles sont parties »)
  • avec avoir → s'accorde avec le COD si placé avant (« les fleurs qu'il a cueillies »)
  • avoir sans COD avant → invariable (« Elles ont mangé »)
⚠ Cherche le COD : « Les pommes que j'ai mangées » (COD = que = pommes, avant → accord).
✅Accords de base
  • Sujet-verbe : le verbe s'accorde avec le sujet (attention au sujet inversé / éloigné)
  • Adjectif : genre + nombre du nom (« une histoire passionnante »)
👯Homophones — le top du Brevet
MotAstuce de remplacement
a / àa = avait ; à = préposition
et / estest = était
son / sontsont = étaient
on / ontont = avaient
ce / sese = devant un verbe
ces / sesces = ceux-là ; ses = les siens
leur / leursleur (verbe) invariable
ou / oùou = ou bien
🧐Pièges fréquents
  • c'est / s'est : s'est + verbe pronominal
  • quel(le) / qu'elle : qu'elle = que + elle
  • peu / peut / peux : peut = pouvait
  • la / là / l'a : l'a = le/la + a
✔ En dictée : à chaque homophone, fais le test de remplacement dans ta tête.
4

Les figures de style

images · oppositions · insistance · sonorités
🖼️Images
  • Comparaison : outil (comme, tel, semblable à). « fort comme un lion »
  • Métaphore : image sans outil. « cet homme est un lion »
  • Personnification : une chose agit comme un humain
  • Allégorie : idée abstraite rendue concrète
📈Exagération & atténuation
  • Hyperbole : exagération (« mourir de faim »)
  • Litote : dire moins pour suggérer plus (« il n'est pas méchant »)
  • Euphémisme : adoucir (« il nous a quittés »)
  • Gradation : intensité croissante
⚔️Oppositions
  • Antithèse : deux idées opposées (« la vie / la mort »)
  • Oxymore : deux mots contradictoires côte à côte (« un silence assourdissant »)
  • Chiasme : croisement (AB / BA)
🔂Insistance & substitution
  • Anaphore : même mot répété en début de phrases/vers
  • Énumération / accumulation
  • Métonymie : la partie pour le tout (« boire un verre »)
  • Périphrase : « la Ville Lumière » = Paris
🎵Sonorités
  • Allitération : répétition de consonnes (« Pour qui sont ces serpents… » → s)
  • Assonance : répétition de voyelles
✔ Pour l'effet : nomme la figure puis explique ce qu'elle produit (insister, choquer, embellir…).
5

Les registres (tonalités)

l'émotion que le texte cherche à provoquer
😀Faire rire / sourire
  • Comique : amuser (gestes, mots, situations, caractères)
  • Ironique : dire le contraire de ce qu'on pense
  • Satirique : se moquer pour critiquer
😢Émouvoir
  • Pathétique : susciter la pitié, l'émotion
  • Tragique : destin fatal, mort inévitable
  • Lyrique : exprimer ses sentiments (je, amour, nature)
⚡Impressionner / convaincre
  • Épique : amplifier l'exploit, le héros
  • Fantastique : doute entre réel et surnaturel
  • Polémique : attaquer vivement une idée
  • Didactique : instruire, expliquer
🔎Comment l'identifier ?

Repère le champ lexical dominant, la ponctuation (! ? …), les figures et le thème.

Ex : beaucoup de « ! », d'hyperboles et un héros invincible → registre épique.

Analyse du récit

narrateur · paroles rapportées · structure
👁️Narrateur & points de vue
  • Interne : on voit par les yeux d'un personnage (ses pensées)
  • Externe : caméra neutre, on ne sait rien des pensées
  • Omniscient (zéro) : le narrateur sait tout, partout
Narrateur « je » (interne au récit) ≠ « il » (extérieur).
💬Les paroles rapportées
TypeIndice
Directguillemets, tirets, « il dit : »
Indirectsubordonnée (« il dit qu'il… »)
Indirect librefondu dans le récit, sans « que »
🪜Le schéma narratif
  1. Situation initiale
  2. Élément perturbateur
  3. Péripéties
  4. Dénouement
  5. Situation finale
⏩Le rythme du récit
  • Scène : on prend le temps (dialogue)
  • Sommaire : on résume vite
  • Ellipse : on saute un moment
  • Pause : description
  • Analepse = retour en arrière · Prolepse = anticipation

Lexique & vocabulaire

sens · niveaux de langue · formation des mots
🌾Champ lexical

Ensemble de mots se rapportant à un même thème.

Ex : mer, vague, écume, marée, large → champ lexical de la mer.
🎭Sens & connotation
  • Dénotation : sens neutre du dictionnaire
  • Connotation : ce que le mot évoque (positif/négatif)
  • Sens propre / sens figuré
Ex : « renard » → propre (animal) / figuré (personne rusée).
🎚️Niveaux de langue
NiveauExemple
Familierbagnole, fric
Courantvoiture, argent
Soutenuautomobile, fortune
🧬Formation & familles de mots
  • Préfixe + radical + suffixe (in-/-able)
  • Synonymes / antonymes
  • Homonymes (verre/vert) · paronymes (conjecture/conjoncture)
  • Racines : graph (écrire), chrono (temps), bio (vie)…
8

Poésie & versification

compter les syllabes · rimes · strophes · rythme
🔢Compter les syllabes
  • Alexandrin = 12 syllabes · décasyllabe = 10 · octosyllabe = 8
  • Le -e muet compte sauf en fin de vers ou devant une voyelle
Ex : « Je / fais / sou / vent / ce / rê / ve / é / tran / ge / et / pé / né / trant » → on découpe syllabe par syllabe.
🎶Les rimes

Disposition :

  • Plates / suivies : AABB
  • Croisées : ABAB
  • Embrassées : ABBA

Qualité : pauvre (1 son), suffisante (2), riche (3+).

🧱Les strophes
  • Distique (2 vers) · tercet (3) · quatrain (4)
  • Sonnet : 2 quatrains + 2 tercets (14 vers)
🎼Le rythme
  • Césure : coupe au milieu de l'alexandrin (6//6), les deux moitiés = hémistiches
  • Enjambement : la phrase déborde sur le vers suivant
  • Rejet / contre-rejet : un mot isolé en début/fin de vers
✔ Relie toujours la forme (rythme, sonorité) au sens (émotion, idée).

Phrases, voix passive & énonciation

types · formes · actif/passif · subordonnées · énonciation
❗Les 4 types de phrase
TypeRôle · signe
Déclarativedéclare un fait ( . )
Interrogativepose une question ( ? )
Injonctiveordre, conseil ( . ou ! )
Exclamativeexprime un sentiment ( ! )
➖Les formes de phrase
  • Affirmative / négative (ne… pas / plus / jamais / rien / personne)
  • Neutre / emphatique : mise en relief « C'est… que », « Moi, je… »
⚠ La négation s'écrit en deux mots : « ne… pas ». À l'oral le « ne » disparaît, pas à l'écrit !
🔄Voix active ↔ voix passive
Active : le sujet fait l'action. Passive : le sujet subit l'action.

Transformation : le COD devient sujet, le sujet devient complément d'agent (« par… »). Le verbe = être + participe passé.

Actif : « Le chat mange la souris. »
Passif : « La souris est mangée par le chat. »
✔ Le participe passé s'accorde toujours avec le sujet au passif (être).
🔗Subordonnées circonstancielles
SensConjonctions
Tempsquand, lorsque, dès que
Causeparce que, puisque, comme
Conséquencesi bien que, de sorte que
Butpour que, afin que
Conditionsi, à condition que
Oppositionbien que, alors que, même si
🗣️L'énonciation

Qui parle ? À qui ? Où ? Quand ? Les indices d'énonciation : pronoms (je/tu/nous), repères de lieu/temps (ici, hier, maintenant), temps verbaux.

Modalisation = marques du jugement de celui qui parle : adverbes (heureusement, sans doute), verbes d'opinion (je crois), vocabulaire mélioratif/péjoratif.

Théâtre & argumentation

genres & outils pour analyser un texte
🎭Le vocabulaire du théâtre
  • Réplique : ce que dit un personnage · tirade : longue réplique
  • Monologue : seul en scène · aparté : à part, pour le public
  • Didascalies : indications de mise en scène (souvent en italique)
  • Acte > scène · scène d'exposition (début) · dénouement (fin)
🎬Comprendre une pièce
  • Quiproquo : malentendu entre personnages
  • Coup de théâtre : retournement brutal
  • Double énonciation : le personnage parle à un autre et l'auteur s'adresse au spectateur

Genres : comédie (faire rire) · tragédie (destin, mort) · drame.

💬Le texte argumentatif
  • Thèse : l'opinion défendue
  • Argument : une idée qui prouve la thèse
  • Exemple : un cas concret qui illustre l'argument
Ex : Thèse « il faut lire » → argument « ça enrichit le vocabulaire » → exemple « en lisant Hugo, j'ai appris… ».
🔌Les connecteurs logiques
SensConnecteurs
Additionde plus, par ailleurs, en outre
Causecar, en effet, parce que
Conséquencedonc, ainsi, c'est pourquoi
Oppositionmais, cependant, néanmoins
Concessioncertes… mais, bien que
🎯Convaincre ou persuader ?
  • Convaincre : par la raison (arguments logiques)
  • Persuader : par les émotions (sentiments, registre pathétique)
✔ Dans un texte d'idées : repère la thèse, souligne les connecteurs, distingue argument et exemple.

Méthodes-types — les 3 épreuves

compréhension · dictée · rédaction
🔎 Répondre à une question de compréhensionMéthode
Règle d'or : une réponse = j'affirme + je cite + j'explique.
1
Je réponds à la question par une phrase claire (je reprends les mots de la question).
2
Je cite le texte entre guillemets (« … » ligne X) pour prouver.
3
J'interprète : j'explique ce que la citation montre, je nomme la figure/le procédé.
✔ Jamais de citation seule, jamais d'affirmation sans preuve. Cite court et précis.
✍️ Réussir la dictéeMéthode
1
Je repère les sujets et j'accorde chaque verbe.
2
À chaque homophone (a/à, et/est, ses/ces…), je fais le test de remplacement.
3
Je vérifie les participes passés (être → sujet ; avoir → COD avant ?).
✔ À la relecture finale : accords dans les groupes nominaux (déterminant + nom + adjectif).
📝 La rédaction — sujet de réflexionMéthode
But : défendre un point de vue de façon organisée (≈ 2 pages).
1
Introduction : j'amène le sujet + j'annonce ma position.
2
Développement : 2-3 arguments, chacun avec un exemple (lecture, actualité, vécu).
3
Je relie avec des connecteurs : d'abord, de plus, cependant, en effet, par conséquent…
✔ Conclusion : je résume et j'élargis. Soigne l'orthographe : elle est notée !
🎨 La rédaction — sujet d'inventionMéthode
1
Je respecte toutes les consignes (qui, où, quand, type de texte, registre).
2
J'assure la cohérence avec le texte de départ (mêmes personnages, même temps du récit).
3
J'enrichis : descriptions, dialogues bien ponctués, figures de style, vocabulaire précis.
✔ Je relis pour la ponctuation du dialogue et les temps (imparfait / passé simple).
